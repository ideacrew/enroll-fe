Will this person file taxes for 2021?

We need to know about your tax filing plans, because if you qualify for monthly premium reductions to help pay for private health insurance, you must file taxes for the year you have coverage.
Answer yes if you plan to file federal taxes for the year you’ll have health coverage, even if you haven’t filed in the past.
Answer no if you don't plan to file federal taxes for the year you’ll have health coverage, even if you haven’t filed in the past.

Will this person be filing jointly?

If you’re married and file your federal taxes jointly, select ‘yes’.
If you don’t plan to file taxes, or plan to file with any other status, select ‘no’.
